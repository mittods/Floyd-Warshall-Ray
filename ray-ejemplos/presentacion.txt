¿Qué es Ray?
Ray es un framework de Python para ejecutar tareas de forma paralela y distribuida.
Permite:
Ejecutar funciones Python en paralelo.
Usar múltiples CPUs.
Usar GPUs si están disponibles.
Crear flujos de trabajo para IA, simulación, optimización y análisis masivo.
Idea clave: Ray transforma funciones normales de Python en tareas remotas.

10 / 22Ray y asignación de recursos
Ray puede trabajar de dos formas:
Automática: detecta CPUs y GPUs disponibles.
Manual: el programador indica cuántos recursos necesita cada tarea.
@ray.remote(num_cpus=2)
def tarea_cpu():
...
@ray.remote(num_gpus=1)
def tarea_gpu():
...
Importante: Ray no obliga a usar GPU. La función debe usar una librerı́a compatible, por
ejemplo PyTorch o TensorFlow.
